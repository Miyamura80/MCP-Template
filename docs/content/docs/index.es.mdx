MCP Template te ofrece tres formas de interactuar con el mismo conjunto de herramientas:

- **CLI** - Ejecuta comandos en tu terminal
- **MCP** - Conecta asistentes de IA como Claude directamente
- **REST API** - Llama a endpoints por HTTP con autenticacion

Elige la interfaz que mejor se adapte a tu flujo de trabajo, o usa las tres.

## Instalacion

import { Step, Steps } from 'fumadocs-ui/components/steps';

<Steps>
<Step>

### Instalar desde el codigo fuente

```bash
git clone https://github.com/Miyamura80/MCP-Template.git
cd MCP-Template
uv sync
```

</Step>
<Step>

### Verificar la instalacion

```bash
uv run mycli --version
uv run mycli --help
```

</Step>
<Step>

### Probar un comando

```bash
uv run mycli greet Alice
# Hello, Alice!
```

</Step>
</Steps>
