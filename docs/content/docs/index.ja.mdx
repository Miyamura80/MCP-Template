MCP Template は同じツールセットに対して3つのインターフェースを提供します:

- **CLI** - ターミナルでコマンドを実行
- **MCP** - Claude などの AI アシスタントを直接接続
- **REST API** - 認証付きで HTTP エンドポイントを呼び出し

ワークフローに合ったインターフェースを選択するか、3つすべてを使用できます。

## インストール

import { Step, Steps } from 'fumadocs-ui/components/steps';

<Steps>
<Step>

### ソースからインストール

```bash
git clone https://github.com/Miyamura80/MCP-Template.git
cd MCP-Template
uv sync
```

</Step>
<Step>

### インストールを確認

```bash
mycli --version
mycli --help
```

</Step>
<Step>

### コマンドを試す

```bash
mycli greet Alice
# Hello, Alice!
```

</Step>
</Steps>
